Overview

A professional cryptocurrency trading portfolio website featuring:

  • Responsive one-page design
  • Interactive trading charts
  • Course marketplace with crypto payments
  • Dark/light mode toggle

[Website Screenshot]

Key Features

Core Functionality

Trading Charts

Integrated TradingView widgets for BTC/ETH with real-time data

Course Marketplace

Multi-cryptocurrency payment system (ETH, BTC, USDT, SOL)

Dark Mode

Theme toggle with localStorage persistence

Technical Highlights

  • Intersection Observer API for scroll animations
  • Dynamic QR code generation for payments
  • Responsive design with CSS Grid/Flexbox
  • Form validation and submission handling

Installation

  1. Clone the repository: 
    git clone https://github.com/Gibsonodoka/selenloveslex.git
  2. Navigate to project directory: 
    cd maindir
  3. Open in browser: 
    Open index.html in your preferred browser
Note: No build process required - works as static site

Configuration

Payment Addresses

Edit education.js with your cryptocurrency addresses:

addresses: {
    eth: "YOUR_ETH_ADDRESS",
    btc: "YOUR_BTC_ADDRESS", 
    usdt: "YOUR_USDT_ADDRESS",
    sol: "YOUR_SOL_ADDRESS"
}

API Keys (Optional)

For enhanced functionality:

  • CoinGecko API for market data
  • Etherscan API for gas prices

Customization

Changing Colors

Edit CSS variables in main.css:

:root {
    --primary-color: #6c5ce7;
    --secondary-color: #00cec9;
    --bg-color: #ffffff;
    /* ... */
}

Adding Courses

Modify the courses object in education.js:

const courses = {
    newCourse: {
        title: "New Course Title",
        prices: {
            eth: "0.1 ETH",
            // other currencies...
        },
        addresses: {
            // addresses...
        }
    }
    // ...existing courses
}

Deployment

Static Hosting

Deploy to any static hosting service:

Netlify

  1. Drag-and-drop the folder
  2. Enable automatic deploys from GitHub

Vercel

  1. Import GitHub repository
  2. Select framework as "Static"
